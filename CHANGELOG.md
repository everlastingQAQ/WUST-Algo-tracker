# Changelog

本项目使用语义化版本管理：`v主版本.次版本.修订版本`。

- 修订版本：修复 bug、小接口补充、不改变核心功能，例如 `v1.1.2 -> v1.1.3`。
- 次版本：新增一组完整功能或数据能力，例如 `v1.1.2 -> v1.2.0`。
- 主版本：大规模架构调整、数据库模型重构或接口不兼容，例如 `v1.x.x -> v2.0.0`。

## v1.1.2

- 新增队长转移接口 `/v1/user/team/owner/transfer`。
- 保留队员退出团队接口 `/v1/user/team/leave`。
- 保留队长解散团队接口 `/v1/user/team/disband`。
- 团队权限形成创建、邀请、管理、转移、退出、解散闭环。
- README 补充 v1.1.2 功能说明和团队接口说明。

## v1.1.1

- 修复成就、团队、消息、统计 UI 对应的后端接口配合问题。
- 完善系统消息、团队邀请、私信和群发消息相关接口说明。

## v1.1.0
